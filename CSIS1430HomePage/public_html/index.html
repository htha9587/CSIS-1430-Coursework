CSIS 1430 Reflection

For CSIS 1430, I learned about web development for the first time. In terms of skills, HTML5, CSS3, Javascript, and Bootstrap were used to create client-side only web pages. This course was an excellent introduction to web development, and the methodologies behind it.

At first we started at basic HTML5 for our home page, resume, and Image Map, but then used CSS to style our web pages. Later, we used Javascript functions to handle the logic and interactivity of our code. This was useful for making random outcomes for the Guessing Game Assignment, with the help of bitwise operators for the Tic-Tac-Toe game. The last three assignments used the CSS Bootrap framework to create web pages with an emphasis on mobile site functionality, and user-friendly design.

At the end of the course, I did feel I learned a great deal that could be expanded upon even more. What served me well the most in this class was my eye for good web design. I would spend some amount of time for each assignment, getting it to look as best as possible. As a result, I feel very proud. Not just what I created, but of how much I learned in the process. Everything about this course was beneficial in one way or another, and it's got me even more excited for what's to come in CSIS 2440.
